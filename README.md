# BuildFlow CRM

AI-powered lead, proposal, email, and pipeline CRM for construction businesses.

This portfolio project is built as a modular monolith:

- Backend: Django REST Framework
- Frontend: React + Vite
- Database: PostgreSQL
- AI: Gemini with mock fallback
- Email: Gmail SMTP with mock-safe demo mode
- WhatsApp: copy-only in V1; direct sending is future scope

See [PROJECT_SETUP.md](PROJECT_SETUP.md) for local setup, PostgreSQL instructions, demo users, and run commands.
